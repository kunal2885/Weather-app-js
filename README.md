# Weather App 

A simple and responsive **Weather Application** built using **Vanilla JavaScript**.  
The app fetches real-time weather data from the **OpenWeatherMap API** and displays current weather information for a searched city.

##  Features

- Search weather by city name
- Displays temperature in Celsius
- Shows humidity and wind speed
- Dynamic weather icons based on conditions
- Handles invalid city names and empty input
- Supports search via button click
- Responsive and clean UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

## How It Works

1. User enters a city name
2. JavaScript validates the input
3. Weather data is fetched using Fetch API with async/await
4. UI updates dynamically based on API response
5. Errors are handled gracefully
